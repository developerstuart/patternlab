# Pattern Lab consumer template

This template demonstrates how to consume `@yourorg/patternlab-core` from another repository.

## Setup

1. Install core package:
   ```bash
   npm install
   ```
2. Build once:
   ```bash
   npm run build
   ```
3. Run dev server:
   ```bash
   npm run dev
   ```

## Files

- `patternlab.config.json`: local consumer configuration
- `src/components/`: consumer-owned components
- `src/data/`: consumer global data
- `src/assets/`: consumer assets
- `plugins/example-plugin.mjs`: local extension plugin
